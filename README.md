# 📊 Digital Study Behaviour Analyzer

A Python-based desktop application to track, analyze, and improve study habits using data visualization, goal tracking, and performance insights.

---

## 📌 Overview

Digital Study Behaviour Analyzer is a GUI application built with CustomTkinter that allows users to log study hours, analyze their performance, and visualize progress through interactive graphs.

It includes a secure login system and personalized dashboards for each user.

---

## 🚀 Features

### 🔐 User Authentication
- Register and login system
- Secure password hashing (SHA-256)
- Multi-user support

### 📝 Study Tracking
- Add study records (Date, Subject, Hours)
- View data in table format
- Delete all or specific records

### 📊 Data Visualization
- Bar graph (subject-wise hours)
- Pie chart (distribution)
- Weekly trend graph

### 🎯 Goal Tracking
- Set goals for subjects
- Track progress with progress bars

### 📉 Dashboard Insights
- Total study hours
- Average hours per day
- Best-performing subject

### 📤 Export
- Export data to Excel file

### 🎨 UI
- Modern interface using CustomTkinter
- Light/Dark mode toggle

---
