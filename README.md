📊 Digital Study Behaviour Analyzer

A Python-based desktop application that helps users track, analyze, and improve their study habits through data visualization, goal tracking, and performance insights.

📌 Overview

Digital Study Behaviour Analyzer is a GUI-based application built using CustomTkinter that allows users to:

Record daily study hours
Analyze subject-wise performance
Visualize progress using graphs
Set and track study goals

It also includes a secure login system with password hashing, making it a personalized and user-specific productivity tool.

🚀 Features
🔐 User System
User registration & login
Secure password storage using SHA-256 hashing
Multi-user support
📝 Study Tracking
Add study records (date, subject, hours)
View all saved data in table format
Delete all or specific records
📊 Data Visualization
📈 Weekly trend graph
📊 Subject-wise bar graph
🥧 Pie chart distribution
🎯 Goal Tracking
Set study goals per subject
Visual progress bars
Real-time performance updates
📉 Analysis Dashboard
Total study hours
Average study hours per day
Best-performing subject
📤 Export
Export study data to Excel
🎨 UI Features
Modern UI using CustomTkinter
Light/Dark theme toggle
🛠️ Tech Stack
Python
CustomTkinter – GUI framework
Pandas – Data handling
Matplotlib – Data visualization
Tkinter (ttk) – Table display
Hashlib – Password security
